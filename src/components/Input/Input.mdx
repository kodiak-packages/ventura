# Input

A Input can be used to let the user insert text using key strokes. A crucial form item for building interactive apps.

## Best practices

- Inputs should be wrapped in a [FormField](/form-field) or should be preceded by another component serving as a label

## Examples

### Basic usage


<Input name="firstname" />


### Input with error


<Input name="username" isInvalid />


### Disabled input

<Input name="username" value="Can't type here" isDisabled />

### Listening for changes

<Input name="logger" onChange={(event) => console.log(`LOGGER: ${event.target.value}`)} />

### Email address

<Input name="email" placeholder="Email address" type="email" />

### Password

<Input name="password" placeholder="Password" type="password" />

### Limited amount of characters


<Input name="countryCode" maxLength={2} className={styles.countryCode} placeholder="XX" />


```css
.countryCode {

```

### Format a serial code

<div>
  {() => {
    const [serialCode, setSerialCode] = React.useState('AAAA-BBBB-CCCC-DDDD');
    const formattedSerialCode = (
      serialCode
        .split('-')
        .join('')
        .match(/.{1,4}/g) || []
    )
      .join('-')
      .toUpperCase();
    return (
      <Input
        name="serialCode"
        value={formattedSerialCode}
        onChange={(event) => setSerialCode(event.target.value)}
        maxLength={19}
      />
    );
  }}
</div>
