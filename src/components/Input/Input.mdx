# Input

A Input can be used to let the user insert text using key strokes. A crucial form item for building interactive apps.

## Best practices

- Inputs should be wrapped in a FormField or should be preceded by another component serving as a label // TODO Link to FormField

### Format a serial code

<Playground>
  {() => {
    const [serialCode, setSerialCode] = useState('AAAA-BBBB-CCCC-DDDD');
    const formattedSerialCode = (
      serialCode
        .split('-')
        .join('')
        .match(/.{1,4}/g) || []
    )
      .join('-')
      .toUpperCase();
    return (
      <Input
        name="serialCode"
        value={formattedSerialCode}
        onChange={(event) => setSerialCode(event.target.value)}
        maxLength={19}
      />
    );
  }}
</Playground>
