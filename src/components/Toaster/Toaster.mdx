# Toaster

Toaster allow you to display status messages.

## Examples

### Basic usage


<Button onClick={(event) => toaster.error('Error toaster')}>Error toaster</Button>
<Button onClick={(event) => toaster.success('Success toaster')}>Success toaster</Button>


### Non closable toaster


<Button onClick={(event) => toaster.error('Error toaster', { isClosable: false })}>Click me</Button>


### Custom toaster duration


<Button onClick={(event) => toaster.success('I\'m here for a looong time.', { duration: 60 })}>Click me</Button>
