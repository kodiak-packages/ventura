# Checkbox

A checkbox can be used enable an optional setting or feature.

## Best practices

- Checkboxes should be wrapped in a [FormField](/form-field) or should be preceded by another component serving as a label

## Examples

### Basic usage

<Playground>
  <Checkbox label="Enable me" name="enabled" />
</Playground>

### Checked state

<Playground>
  <Checkbox label="Disable me" name="enabled-2" defaultValue={true} />
</Playground>

### Checkbox with description

<Playground>
  <Checkbox label="Enable me" name="enabled-3" description="This feature can be enabled" />
</Playground>

### Disabled checkbox

<Playground>
  <Checkbox label="You can't enable me" name="enabled-4" isDisabled />
</Playground>

### Listening for changes

<Playground>
  <Checkbox label="Log" name="enabled-5" onChange={(event) => console.log(`LOGGER: ${event.target.checked}`)} />
</Playground>
