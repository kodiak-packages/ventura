# Alert

Alerts allow you to display error messages relevant to an entire form or application.

## Examples

### Basic usage

<Alert intent='error' message='Something went wrong.' />

### Different intent


<Alert intent='success' message='This went great!' />
<br />
<Alert intent='warning' message='I would like to warn you.' />
<br />
<Alert intent='error' message='Something went wrong.' />
